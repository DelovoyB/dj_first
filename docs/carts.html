carts.utils.get_user_carts(request)[source]

Get a user’s carts from the database.

If the user is authenticated, we query the database with the user’s id. If the user is not authenticated, we query the database with the session key. If the session key doesn’t exist yet, we create it.

Returns a QuerySet of Cart objects related to the user.

test_create_cart() → None[source]

Test that a cart can be created with valid user, product, quantity, session key and created timestamp.

The full_clean method is called to ensure that the cart is valid according to the model’s field definitions. The save method is then called to persist the cart to the database.

The test then checks that the cart has been created with the expected user, product, quantity, session key and created timestamp.
